<h1 align="center"> Lista de tareas sencilla owo </h1>

## :heart:Descripción del proyecto
Este proyecto permite llevar un control de las tareas diariares, dividiendolas en eststus de pendientes y completadas.

## :heart:Funcionalidades del proyecto
- `Agregar tareas`: Se agregan tareas a una lista de pendientes
- `Validación`: Valida que la tarea no este vacia
- `Completar tareas`: Al presionar el botón completar en una tarea, se mueve al área de completadas
- `Eliminar tareas`: Se remueve la tarea por completo de la lista

## :heart:Instrucciones de uso
- `Inicializar`: Ejecutar el archivo index.html
- `Agregar tarea`: Escribir la tarea en el cuadro principal y agregarla con el botón +
- `Completar tarea`: Presionar el botón O junto a la tarea
- `Eliminar tarea`: Presionar el botón X junto a la tarea

## :heart:Tecnologías utilizadas
- `JavaScript`: Funcionalidades basicas de la aplicación
- `CSS`: Estilos y diseño
